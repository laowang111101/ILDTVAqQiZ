# 书籍管理系统 python1193

## 项目概述

书籍管理系统是一个基于Python语言的在线平台，旨在提供便捷的书籍信息浏览和管理功能。系统分为前台用户交互界面和后台管理界面，支持用户中心、购物车、订单管理以及书籍管理等多种功能。

## 技术栈

### 后端

- 开发语言：Python
- 框架：Django, OpenCV
- Python版本：3.7.7
- 数据库：MySQL 5.7+
- 数据库工具：Navicat 11+

### 前端

- NodeJS
- Vue
- HTML

## 功能模块

### 前台功能

- 用户中心
- 我的购物车
- 我的订单
- 商品分类

### 后台功能

- 书籍管理
- 添加书籍
- 订单管理
- 退订管理
- 用户管理

## 系统角色

系统区分两种角色：普通用户和超级管理员。

- 普通用户：可浏览书籍信息、添加到购物车、下订单等。
- 超级管理员：拥有权限登录后台，管理网站信息、书籍、订单和用户。
